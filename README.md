# next-auth-boilerplate

A boilerplate to save time on setting up:

- Next.js
- Tailwind
- Prisma
- Next-auth
- Sqlite

Clone it into your (empty) local directory

`git clone https://github.com/eesazahed/next-auth-boilerplate.git .`

then install the node modules 

`npm i`


then run it 

`npm run dev`
